# Synthetic-Money
______
## A website for generating synthetic financial data based on russian financial information from public sources.
To read about logic, weights in the code, etc. open file called 'logic_drafts.md' or 'logic_drafts.pdf' in 'backend/research'.

User can input some information such as size of the data, percent of fraud transactions, format of the final file, etc.

The program will return file based on the requirements and basic logic from the research paper. The format of the final file can be: `xlsx`, `csv`, `parquet`, or `json`.
